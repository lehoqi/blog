详细步骤说明

expand_less
1

检查 Chrome 安装

确保 Google Chrome 已安装在您的电脑上。如果没有安装，请访问 chrome.google.com 下载并安装。

2

设置 Chrome 语言为英语（macOS 必需）

macOS 用户必须先设置语言，否则 Gemini 无法启用。

macOS: 系统设置 → 通用 → 语言与地区 → Chrome 应用语言设为"英语" → 重启 Chrome

Linux/Windows: Chrome 地址栏输入 chrome://settings/languages → 英语设为首选

3

登录美区 Gmail 账号

在 Chrome 中登录您的美区 Gmail 账号，这是启用 Gemini 的关键。

Chrome 点击头像 → 添加账号 → 选择美区账号

info 没有美区账号？前往注册 Gmail →

4

复制并执行安装命令

复制上方命令，打开终端粘贴执行。脚本会自动完成配置修改。

5

重启 Chrome

完全关闭 Chrome 后重新打开，访问 gemini.google.com 即可使用！

help

常见问题 FAQ

expand_less

为什么要设置语言和登录美区账号？

Gemini 的地区检测依赖两个条件：

  • macOS 必须将 Chrome 语言设为英语 - 应用语言影响地区判断
  • 登录美区 Gmail 账号 - 账号区域是主要判断依据

两者缺一不可，仅修改配置文件无法启用。

如何设置 Chrome 语言和切换账号？

设置语言：

  • macOS: 系统设置 → 通用 → 语言与地区 → Chrome 应用语言设为英语
  • Linux/Windows: Chrome 访问 chrome://settings/languages

切换账号：Chrome 点击头像 → 添加账号 → 选择美区账号

支持哪些操作系统？

本工具支持 macOS、Linux、Windows (Git Bash/WSL)。脚本会自动检测系统并应用对应配置。

遇到问题如何还原？

脚本会自动备份原配置文件。如果遇到问题，运行以下命令还原：

bash install.sh --rollback

或使用调试模式查看详细日志：bash install.sh --debug

这安全吗？

脚本是开源的，您可以查看完整源代码。脚本会自动备份原配置文件，并且可以一键还原。但请注意，修改系统配置始终存在一定风险。

如何打开终端

laptop_mac macOS

打开"启动台" → "终端" 或按 Command + 空格 搜索"Terminal"

computer Linux

  • Git Bash: git-scm.com
  • WSL: 在 PowerShell 中运行 wsl
